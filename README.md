# 🌸 MHM Hub Website

A simple and modern Menstrual Health & Hygiene Management (MHM) website project.

This project helps users:

* Track menstrual cycles
* Learn hygiene & health tips
* Find nearby sanitary product locations
* Access a clean and beginner-friendly health platform

# 💻 How to Create Project Folder Using CMD

Open CMD and run:

```cmd
cd %USERPROFILE%\Desktop
mkdir MHM
cd MHM
mkdir css js

type nul > index.html
type nul > login.html
type nul > register.html
type nul > tracker.html
type nul > health.html
type nul > map.html
type nul > css\style.css

```

# 🌐 How to Run the Website

## Method 1 — Direct Browser Open

1. Open project folder
2. Double-click `index.html`
3. Website will open in browser

---

## Method 2 — VS Code Live Server (Recommended)

### Step 1 — Install Live Server Extension

In VS Code:

* Open Extensions
* Search: `Live Server`
* Install extension

### Step 2 — Run Website

Right-click `index.html`

Click:

```bash
Open with Live Server
```

Website will start automatically.
